<h1>Have app</h1>

<p>It plugin does works with applications installed on devices.</p>

<br />

<aside>⚠️ For now, it does works only for Android.</aside>

<br />

<h2>Methods</h2>

`getPackage`  -  get application by `GetApplication` class.

<br />

<h2>Install</h2>

```bash
flutter pub add have_app
```
